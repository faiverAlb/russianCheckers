# Russian Checkers
Russian Checkers game implementation: Player vs Simple MinMax Robot with Alpha-Beta pruning 

# Features:
1. Choose side
2. User can select "thinking" time for robot to calculate must reasonable movement
3. Undo/Redo
4. Simple unit tests
